NODE SHAPE & COLOR RULES (use MUTED colors only):

- Process / Action (assignments, calculations, general statements):
    shape=box, style="filled,rounded", fillcolor="#e3f2fd", color="#1976d2", fontcolor="#000000"

    shape=diamond, style=filled, fillcolor="#fff9c4", color="#fbc02d", fontcolor="#000000"

- Function Call / Subroutine:

LAYOUT RULES:
- Use `rankdir=TB` for top-to-bottom layout.
- Graph settings: bgcolor="transparent", fontname="Arial", splines=ortho, nodesep=0.7, ranksep=0.9
- Nodes: fontname="Arial", fontsize=12, penwidth=1.2
- Edges: color="#555555", fontname="Arial", fontsize=11, fontcolor="#333333"
- Wrap long labels at ~25 chars using \n in the label string.

EDGE / TRANSITION RULES:
- Label Yes/No or True/False on decision branches clearly.
- Loop back-edges must point back to the correct decision node.
- Normal flow edges have empty label "".

    fontname="Arial";

    Init    [label="result = 1\ni = 1", shape=box,           style="filled,rounded", fillcolor="#e3f2fd", color="#1976d2", fontcolor="#000000"];
    Check   [label="i <= n?",          shape=diamond,        style=filled, fillcolor="#fff9c4", color="#fbc02d", fontcolor="#000000"];
    Calc    [label="result *= i\ni++", shape=box,           style="filled,rounded", fillcolor="#e3f2fd", color="#1976d2", fontcolor="#000000"];

    End     [label="End",              shape=ellipse,        style=filled, fillcolor="#d4edda", color="#28a745", fontcolor="#000000"];

    Start -> Init;
    Init  -> Check;
    Check -> Calc  [label="Yes"];
    Calc  -> Check [label="Loop"];
    Check -> Print [label="No"];
    Print -> End;
}
